## 📌 项目概览
本项目是一个基于数字信号处理（DSP）的被动雷达与运动检测系统。系统利用已知通信信号作为参考源（Reference Signal），接收目标反射的监视信号（Surveillance Signal）。通过一系列时频域信号处理算法，精准提取目标的**距离（时延 $\tau$）**与**速度（多普勒频移 $f_d$）**信息，最终生成直观的 Time-Doppler 谱图（时延-多普勒谱）。

## 🧠 核心算法与工程优化
本项目在标准互相关运算的基础上，针对海量采样数据进行了深度的工程优化：

1. **数字下变频 (DDC) 与 滤波降噪**: 
   将高频信号通过复频移技术平滑搬移至基带，并设计了高阶 Butterworth 低通滤波器（9MHz 截止频率），大幅降低后续计算的信噪比干扰，提取出纯净的包络信息。
2. **动态搜索范围模糊函数 (`ambiguity_fast.m`)**:
   突破了传统全局遍历算法的算力瓶颈。在首次粗搜锁定峰值后，算法能够**动态自适应缩小频率窗口（如 $-30\text{Hz}$ ~ $+30\text{Hz}$）与时移搜索域**。在保证精度的同时，极大缩短了 20 段数据的整体运算时间。

## 📁 核心代码结构
本项目采用模块化设计，包含以下 4 个核心 MATLAB 脚本：

* `modulation.m`: **信号预处理模块**。包含 DDC 变频、低通滤波以及快速傅里叶变换（FFT）的频域与时域对比可视化。
* `ambiguity.m`: **基础模糊函数模块**。采用全量静态搜索范围进行二维互相关计算（用于与快速算法进行精度和性能对照）。
* `ambiguity_fast.m`: **快速模糊函数模块**。核心优化代码，引入动态搜索域机制，大幅提升运行效率。
* `project2_main_analysis.m`: **系统总控脚本**。整合数据加载、DDC 处理、匹配度 3D 渲染，并最终生成完整的 Time-Doppler 谱图及算法耗时柱状图。

## 🚀 快速开始
1. 克隆本仓库到本地。
2. 确保你的 MATLAB 环境已准备就绪（建议包含 Signal Processing Toolbox）。
3. 将包含观测数据的 `.mat` 文件放置于对应的 `data/` 目录下（请根据代码中的 `basePath` 自行调整数据路径）。
4. 直接运行 `project2_main_analysis.m`，即可自动执行 20 段数据(相关数据来源于老师在此不方便公开)的批量处理，并生成相应的 3D 频谱与耗时分析图。
